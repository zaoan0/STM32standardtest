# STM32 直线循迹小车项目

## 项目描述

基于 STM32F103C8 的直线循迹小车系统，使用5路灰度传感器进行线路检测，电机PWM控制速度调节。v1.1.0 新增 MPU6050 陀螺仪，用于脱线恢复时的方向控制。

## 硬件配置

| 组件 | 说明 |
|------|------|
| MCU | STM32F103C8 |
| 灰度传感器 | 5路（GPIOB: Pin 3/4/5/12/13） |
| 电机驱动 | 双电机PWM控制（TIM3: PA6/PA7） |
| 陀螺仪 | MPU6050（软件I2C: PB8-SCL, PB9-SDA） |
| 串口 | USART1（PA9-TX，PA10-RX）115200bps |

## 引脚分配

| 引脚 | 功能 | 模块 |
|------|------|------|
| PA6 | 左轮 PWM | motor (TIM3_CH1) |
| PA7 | 右轮 PWM | motor (TIM3_CH2) |
| PA9 | USART1 TX | debug |
| PA10 | USART1 RX | debug |
| PB0 | 左轮 IN1 | motor |
| PB1 | 左轮 IN2 | motor |
| PB3 | 灰度 OUT1（最右） | grayscale |
| PB4 | 灰度 OUT2（右中） | grayscale |
| PB5 | 灰度 OUT3（中间） | grayscale |
| PB8 | MPU6050 SCL | mpu6050（软件I2C） |
| PB9 | MPU6050 SDA | mpu6050（软件I2C） |
| PB10 | 右轮 IN3 | motor |
| PB11 | 右轮 IN4 | motor |
| PB12 | 灰度 OUT4（左中） | grayscale |
| PB13 | 灰度 OUT5（最左） | grayscale |
| PB14 | 电机 STBY | motor |
| PB15 | 按钮（上拉输入） | main |

## 功能模块

### 核心模块
- **motor.c/h** - 电机初始化和PWM控制
- **grayscale.c/h** - 灰度传感器读取
- **mpu6050.c/h** - MPU6050 软件I2C驱动（v2.0.0 新增）
- **main.c** - 主循迹逻辑

### 支持模块
- **pid.c/h** - PID控制算法（预留）
- **encoder.c/h** - 编码器读取（预留）
- **debug.c/h** - 串口调试支持
- **delay.c/h** - 延时函数（DWT微秒延时 + SysTick毫秒延时）

## 版本历史

### v1.1.0（2026-05-01）— 当前版本 ✅

**核心机制：MPU6050 陀螺仪辅助脱线恢复**

脱线后根据灰度传感器记录的方向，使用陀螺仪控制转向角度主动寻线。

**脱线恢复流程：**
```
灰度传感器检测到线 → 正常循迹（差速转向）
    ↓
全部传感器脱线
    ↓
判断上次传感器方向：
  - SL/LM = 1 → 左转 30° 找线
  - SR/RM = 1 → 右转 30° 找线
  - 无方向信息 → 保持直行
    ↓
陀螺仪实时积分 yaw 角，P 控制器平滑转向目标角度
    ↓
灰度传感器重新检测到线 → 恢复正常循迹
```

**关键技术：**
- 软件 I2C 驱动 MPU6050（PB8/PB9），不占用硬件 I2C 资源
- 选择性积分：角速度 > 1 dps 时才积分，静止时 yaw 不漂移
- 启动校准：上电静止 1.25 秒，采 500 次取平均作为零偏
- DWT 微秒延时：基于 Cortex-M3 周期计数器，不受编译器优化影响

**参数配置：**
```c
#define SPD_BASE        200   // 基础速度
#define SPD_SLIGHT       75   // 微调转向速度差
#define SPD_STRONG      150   // 强转向速度差
#define GYRO_SCALE    0.01527f // 陀螺仪 LSB→dps (1/65.5)
#define INTEG_THRESH    1.0f   // 积分角速度门限 (度/秒)
#define LOST_ANGLE     30.0f   // 脱线最大转向角度 (度)
#define LOST_KP         4.0f   // 脱线转向比例系数
```

**版本特点：**
- ✅ 脱线后智能判断方向，主动寻线
- ✅ 陀螺仪控制转向角度，防止转圈
- ✅ 选择性积分，静止不漂移
- ✅ 软件 I2C，保留全部硬件资源
- ⚠️ 长时间旋转仍会有积分漂移
- ⚠️ 需要 4.7k 上拉电阻（部分 MPU6050 模块自带上拉）

### v1.0.1（2026-04-26）— 历史版本

**核心机制：双阶段脱线恢复**

脱线后通过 `STABLE_COUNT` 稳定计数确认信号可靠再恢复，防止传感器抖动误触发。

**版本特点：**
- ✅ 稳定计数防止传感器抖动误触发恢复
- ✅ 脱线时保持上次转向速度，主动寻线
- ⚠️ 恢复需稳定持续约0.5s，响应稍慢

---

### v1.0.0（2026-04-26）— 历史版本

**核心机制：编译期可配置脱线模式**

通过 `LOST_MODE` 宏在编译时选择脱线策略。

**版本特点：**
- ✅ 代码简洁，逻辑清晰
- ✅ 两种脱线策略可通过宏一键切换
- ⚠️ 无稳定计数，传感器抖动时可能误触

## 硬件接线

### MPU6050 接线

| MPU6050 | STM32 | 备注 |
|---------|-------|------|
| VCC | 3.3V | 不可接 5V |
| GND | GND | 共地 |
| SCL | PB8 | 需 4.7k 上拉到 3.3V |
| SDA | PB9 | 需 4.7k 上拉到 3.3V |
| AD0 | GND | I2C 地址 0x68 |

> 部分 MPU6050 模块（如 GY-521）板上已自带上拉电阻，无需外接。

### 电机驱动接线

| 驱动引脚 | STM32 | 功能 |
|----------|-------|------|
| PWMA | PA6 | 左轮 PWM |
| PWMB | PA7 | 右轮 PWM |
| AIN1 | PB0 | 左轮方向 |
| AIN2 | PB1 | 左轮方向 |
| BIN1 | PB10 | 右轮方向 |
| BIN2 | PB11 | 右轮方向 |
| STBY | PB14 | 使能（常高） |

## 使用说明

1. 用 Keil 打开 `stm32standardtest.uvprojx`
2. 编译并下载到 STM32F103C8
3. 上电后小车自动直走 1 秒（启动测试），然后开始循迹
4. 脱线后自动根据方向恢复
