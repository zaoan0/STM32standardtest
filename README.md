# STM32 触摸屏循迹小车 (战舰V3)

基于 **STM32F103ZET6** (正点原子战舰V3) 的循迹小车，配备 4.3" TFT 触摸屏，支持实时调参、状态显示和陀螺仪辅助脱线恢复。

本项目从 [STM32standardtest](https://github.com/zaoan0/STM32standardtest) (F103C8 小板) 演进而来，针对战舰V3 的引脚布局和外设资源做了全面重构。

---

## 硬件配置

| 组件 | 说明 |
|------|------|
| MCU | STM32F103ZET6（战舰V3 核心板，144pin，512KB Flash） |
| LCD | 4.3" 800x480 TFT LCD（NT35510，FSMC 16-bit 接口） |
| 触摸 | GT9147 电容触摸（软件I2C: PB1-SCL, PF9-SDA） |
| 陀螺仪 | MPU6050（软件I2C: **PB1-SCL, PB2-SDA**） |
| 灰度传感器 | 感为 GW-8ARS 红光八路模拟灰度传感器（74HC4051 多路复用） |
| 电机驱动 | 双电机 PWM 控制（TIM3, PA6/PA7, 1kHz） |
| 编码器 | 增量式，TIM2 左轮(PA0/PA1)，TIM4 右轮(PB6/PB7)（预留） |
| 串口 | USART1（PA9-TX, PA10-RX）115200bps |

## 引脚分配

### 电机

| 引脚 | 功能 | 说明 |
|------|------|------|
| PA6 | 左轮 PWM | TIM3_CH1 |
| PA7 | 右轮 PWM | TIM3_CH2 |
| PE2 | 左轮 IN1 | 方向控制 |
| PE3 | 左轮 IN2 | 方向控制 |
| PE4 | 右轮 IN3 | 方向控制 |
| PC0 | 右轮 IN4 | 方向控制 |
| PC3 | 电机 STBY | 使能，常高 |

### 8路灰度传感器（感为 GW-8ARS）

| 引脚 | 功能 | 说明 |
|------|------|------|
| PC4 | AD0 | 地址选择 bit0 |
| PC5 | AD1 | 地址选择 bit1 |
| PC1 | AD2 | 地址选择 bit2 |
| PA4 | OUT | ADC1_CH4 模拟输入 |
| EN | 悬空 | 内部下拉，默认使能 |
| +5V | 电源 | 需 5V 供电（最低 5V，推荐独立供电） |
| GND | 地 | 必须与 MCU 共地 |

> 74HC4051 真值表：AD2:AD1:AD0 = 000~111 选择 CH1~CH8

### 通信接口

| 接口 | 引脚 | 用途 |
|------|------|------|
| MPU6050 I2C | PB1 (SCL), PB2 (SDA) | 软件 I2C（避开 JTAG） |
| GT9147 触摸 I2C | PB1 (SCL), PF9 (SDA) | 软件 I2C |
| USART1 | PA9 (TX), PA10 (RX) | 调试串口 |

### LCD (FSMC，底板直连)

| 引脚 | 功能 |
|------|------|
| PD0,1,4,5,8-10,14,15 | FSMC 数据/控制 |
| PE7-15 | FSMC 数据 |
| PG0 | FSMC A10 (RS) |
| PG12 | FSMC NE4 (片选) |
| PB0 | LCD 背光 |

## 软件功能

### 多页面 UI

| 页面 | 功能 |
|------|------|
| 主页 | 4个大按钮导航：循迹模式、陀螺仪直线、待开发1、待开发2 |
| 循迹页 | 停止/循迹模式切换，Base/Slight/Strong 速度调节，8路传感器实时显示 |
| 陀螺仪页 | 目标角度、速度可调，START/STOP 控制，yaw/输出实时显示 |
| 空白页 | 占位，"敬请期待" |

所有页面支持中文显示（GB2312 编码，16x16 点阵字模）。

### 循迹算法（8路加权平均）

8路模拟传感器计算加权偏差，实现比例转向：

```
weights = {7, 5, 3, 1, -1, -3, -5, -7}
error = Σ(weight[i] * strength[i]) / Σ(strength[i])

pwm_left  = SPD_BASE + error * SPD_STRONG
pwm_right = SPD_BASE - error * SPD_STRONG
```

- 传感器检测黑线时 ADC 值低于阈值（默认 2000）
- 多传感器同时触发时产生连续偏差，过渡平滑
- 参数可通过触摸屏实时调节

### 单陀螺仪直线模式

不依赖灰度传感器，仅用陀螺仪保持直线行驶：

- P 控制保持目标 yaw 角度
- 目标角度和速度可调
- 适合无循迹线的直道段

### 脱线恢复（陀螺仪辅助）

1. 脱线瞬间记录 yaw 角度和最后看到线的传感器位置
2. 根据线消失方向设定目标旋转角 (+/-30 度)
3. P 控制 (Kp=4.0) 旋转到目标角度
4. 找到线后恢复正常循迹

## 编译环境

- **IDE**: Keil uVision5
- **编译器**: ARMCC V5.06 update 5
- **C 标准**: C99
- **优化等级**: -O1
- **预定义宏**: `STM32F10X_HD, USE_STDPERIPH_DRIVER`
- **包含路径**: `.\USER; .\CORE; .\FWLIB\inc`
- **注意**: 含中文的源文件必须保存为 **GB2312** 编码（ARMCC V5 不支持 UTF-8 中文）

---

## 使用方法

1. 用 Keil 打开 `stm32zhanjianstandard.uvprojx`
2. 编译并烧录到战舰V3
3. 上电后触摸屏显示主界面
4. 点击 **循迹模式** 进入循迹页面
5. 点击 **循迹** 按钮开始循迹，**停止** 按钮停止
6. 通过 **[-]/[+]** 调节 Base/Slight/Strong 速度参数
7. 点击 **陀螺仪直线** 进入直线模式，设置角度和速度后点击 **运行中** 启动

## MPU6050 接线

| MPU6050 | STM32 | 备注 |
|---------|-------|------|
| VCC | 3.3V | 不可接 5V |
| GND | GND | 共地 |
| SCL | **PB1** | 软件 I2C |
| SDA | **PB2** | 软件 I2C |
| AD0 | GND | I2C 地址 0x68 |

> 注：MPU6050 引脚从 PB4/PB5 改为 PB1/PB2，保留 JTAG 烧录功能。

## 8路灰度传感器接线

| 传感器 | STM32 | 备注 |
|--------|-------|------|
| +5V | 5V 电源 | 需独立供电，电流 >= 150mA |
| GND | GND | 必须与 MCU 共地 |
| AD0 | PC4 | 地址选择 bit0 |
| AD1 | PC5 | 地址选择 bit1 |
| AD2 | PC1 | 地址选择 bit2 |
| OUT | PA4 | ADC 模拟输入 |
| EN | 悬空 | 内部下拉，默认使能 |

> 感为 GW-8ARS 红光八路模拟灰度传感器，检测高度 4-100mm，推荐 10-20mm。
